# NXTUP (MVP) — App

This is the **comp-first** NXTUP MVP app scaffold (React + TypeScript + Tailwind).

## Prereqs

- Node.js 20+ (or 18+ should work)
- npm / pnpm / yarn

## Run locally

```bash
cd "NXTUP project sandbox/nxtup"
npm install
npm run dev
```

## Notes

- This repo starts with **stubbed auth** (demo user) and mocked data so we can iterate quickly on UI + flows.
- Later milestones will add Supabase Auth + RLS and server persistence.
